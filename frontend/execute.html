How It Works

Your input goes through an 8-stage deterministic pipeline:

  1. Entity Extraction (spaCy NER)
  2. Intent Classification (forced overrides + ML)
  3. Rule Evaluation (permissions & thresholds)
  4. Tool Selection (intent mapping)
  5. Parameter Building (schema-driven)
  6. Schema Validation (JSON Schema)
  7. Tool Execution (MCP transport)
  8. Response Formatting
